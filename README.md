# ungroup

## Intro

A game about temporary alliances.

## Building and Running
**Unix**
1. Clone this repo
2. Install SFML-devel package
3. To configure run:
```
cmake -S . -B build
```
4. To build run:
```
cmake --build build -- -j
```
5. To run game run:
```
./build/src/server/ug-server
```
```
./build/src/client/ug-client
```
